# API of ReacNetGenerator

ReacNetGen.

# class reacnetgenerator.ReacNetGenerator(**kwargs)

Use ReacNetGenerator for trajectory analysis.

# class Status(value)

ReacNetGen status.

The ReacNetGen consists of several modules and algorithms to process the information from the given trajectory.

# draw()

Draw the reaction network.

# report()

Generate the analysis report.

# run()

Process MD trajectory.
